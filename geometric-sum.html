freskhii's articles - The gemoetrical sum

The famous sum $1 + q^1 + q^2 + \ldots + q^n = \frac{q^{n+1}-1}{q-1}$ - noted $S_n$ in this article - is generally proved by expressing $S_{n+1}$ in terms of $S_n$ , $q$ and $n$. I'll present you another approach for $q,n\in \mathbb N$.

Let $S_n := 1 + q + q^2 + \ldots + q^n$.

If $q=1$, $S_n = n+1$.

Let's try something when $q=2$.
In the binary base $S_n = 1 + 2 + \ldots + 2^n = 2^n + \ldots + 2 + 1= (1\ldots11)_2$. (the notation means "in base 2")
But $1\ldots11$ is also the biggest number with n+1 digits of binary base and there's only two of them so there's only $2^{n+1}$ numbers.
Consequently, by indexing each one of them in the ascending order, the biggest number (aka $1\ldots11$) has the index $2^{n+1}$.
$0$ get the index $1$ so the value of the number is the index $-1$. At last the biggest number is equal to $2^{n+1}-1$.

If we try to generalize the method for all $n \ge 2$ and $q \ne 1$, we get :
$S_n = 1 + q + \ldots + q^n=(1\ldots11)_q$ with $n+1$ ones. But the largest number with $n+1$ digit of the base $q$ is made with the digit which value is $q-1$.
Let $k$ bet that digit, we obtain $k\ldots kk = q^{n+1}-1$, i.e. $$(q-1)\times q^n + \ldots + (q-1) \times q + q = q^{n+1}-1 \\ \Leftrightarrow q^n + \ldots + q + 1 = \frac{q^{n+1}-1}{q-1}$$

$\blacksquare$
